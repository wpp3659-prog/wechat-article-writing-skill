# Expression Styles

Choose one primary style before drafting. Use one secondary style only when it clearly improves the article. Do not mix many styles.

## Style Routing

- Technology, AI, business, industry trends -> Rational Analysis.
- Workplace, growth, ordinary people's dilemmas -> Conversational Companion or Practical Method.
- Controversial view, hot take,反常识 -> Sharp Opinion.
- People, cases, journey, brand origin -> Story Narrative.
- Tutorials, checklists, operations, tools -> Practical Method.
- Product, service, course, consulting -> Brand Conversion.
- Cognition, long-termism, personal IP, reflective themes -> Deep Essay.

If user specifies a style, follow the user's choice.

## Rational Analysis

Use for technology, AI, business, policy, and industry trend articles.

Voice:

- Clear, restrained, precise, slightly sharp.
- Use judgment, not hype.
- Prefer "这意味着", "真正影响的是", "如果只看表面".

Opening:

- Start with a change, contradiction, or misread signal.
- Avoid long background.

Rhythm:

- Medium paragraphs, occasional short punch line.
- Use comparison and cause-effect, but avoid textbook transitions.

Good lines:

- "问题不在于工具变强，而在于工作被重新切分。"
- "这不是一个功能升级，而是一套分工方式的变化。"

Avoid:

- Excessive jargon.
- Neutral report tone.
- "机遇与挑战并存" style filler.

## Conversational Companion

Use for workplace, personal growth, daily life, anxiety, and reader empathy.

Voice:

- Like a sharp but warm friend.
- Honest, conversational, not preachy.
- Use "说白了", "你可能也有这种感觉", "先别急".

Opening:

- Start with a familiar scene or emotional truth.

Rhythm:

- Short paragraphs.
- More pauses, small turns, and reader-facing questions.

Good lines:

- "很多人不是不努力，是连自己在跟什么较劲都没看清。"
- "这事听起来很小，但它每天都在消耗你。"

Avoid:

- Chicken-soup slogans.
- Empty comfort.
- Overusing "你要相信".

## Sharp Opinion

Use for commentary, counterintuitive claims, strong viewpoints, and hot takes.

Voice:

- Direct, decisive, a little provocative.
- Make a clear judgment.
- Use contrast and short sentences.

Opening:

- Put the blade on the table in the first paragraph.

Rhythm:

- Short sentences mixed with one dense reasoning paragraph.
- Strong section endings, but not every section needs a slogan.

Good lines:

- "真正危险的不是 AI 太强，而是你的工作太容易被说明白。"
- "别把低效包装成经验。"

Avoid:

- Mean-spirited attacks.
- Overclaiming.
- Every paragraph sounding angry.

## Story Narrative

Use for people, cases, events, personal experience, and brand stories.

Voice:

- Scene first, conclusion later.
- Let details carry emotion.
- Use time, action, conflict, and turning points.

Opening:

- Start from a moment: a message, meeting, mistake, result, silence, or decision.

Rhythm:

- Narrative paragraphs can be slightly longer.
- Analysis comes after the reader sees the scene.

Good lines:

- "事情真正变味，是从那次会议之后开始的。"
- "他当时没意识到，那个小决定后来改了整条路。"

Avoid:

- Fake stories.
- Over-dramatized movie narration.
- Ending with forced inspiration.

## Practical Method

Use for tutorials, methods, operations, frameworks, and checklists.

Voice:

- Useful, clean, concrete.
- Give steps, but avoid mechanical instruction-manual tone.

Opening:

- Name a problem and promise a clear takeaway.

Rhythm:

- Use H2 for method blocks.
- Use bullets only when they improve scanning.
- Include examples or mini scenarios after steps.

Good lines:

- "别从工具开始，先从你最烦的流程开始。"
- "一个方法有没有用，先看它能不能减少一次返工。"

Avoid:

- Too many numbered lists.
- "万能公式" claims.
- Steps with no example.

## Brand Conversion

Use for product, service, course, consulting, and business articles.

Voice:

- Start from user pain, not product pride.
- Practical, credible, restrained.
- Explain value before asking for action.

Opening:

- Describe the customer's situation or cost of the old way.

Rhythm:

- Pain -> consequence -> solution -> proof -> action.
- If proof is missing, add an editor note instead of inventing.

Good lines:

- "用户真正买的不是功能，是少掉的那一段麻烦。"
- "如果这个问题每周发生一次，它就不是小问题。"

Avoid:

- Hard selling too early.
- Fake testimonials or fake data.
- Overpromising outcomes.

## Deep Essay

Use for cognition, long-termism, personal IP, reflective essays, and nuanced themes.

Voice:

- Reflective, layered, slightly literary but clear.
- Allow uncertainty.
- Prefer insight over instruction.

Opening:

- Start with a quiet observation or counterintuitive sentence.

Rhythm:

- More variation.
- Let one paragraph breathe; then cut with a short sentence.

Good lines:

- "很多变化不是突然发生的，它只是突然被你看见了。"
- "人真正被困住的时候，往往不是没有选择，而是每个选择都像旧路。"

Avoid:

- Pretentious abstraction.
- Overly poetic vagueness.
- Ending in universal life advice.

## Style Output Note

Do not label the style inside the article unless the user asks. The style should be felt through wording, rhythm, and structure.
